# Roadmap

## Version 0.1

- Static dashboard
- Local saved tracking IDs
- Refresh on page open
- CSV import and export
- Status filters

## Version 0.2

- Add a backend proxy for a real tracking API
- Store tracking IDs in a database
- Add scheduled refresh
- Add error states for invalid tracking IDs

## Version 0.3

- Add user accounts
- Add customer/order names
- Add WhatsApp-ready customer status messages
- Add delivery delay alerts

## Version 1.0

- Support small seller workflows end to end
- Add multi-courier provider support
- Add team dashboard
- Add production deployment
