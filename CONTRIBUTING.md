# Contributing

PostPilot India is built for small sellers who need a simple bulk tracking workflow.

## Good First Contributions

- Improve the dashboard UI
- Add empty/error/loading states
- Improve CSV parsing
- Add tests for provider logic
- Document a real courier API integration

## Guidelines

- Keep the interface simple and fast.
- Avoid adding dependencies unless they remove real complexity.
- Do not add scraping logic without documenting legal and operational risks.
- Keep README instructions beginner-friendly.
